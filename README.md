# DİJİTAL METİN
Kullanıcının yazdığı metni, #, * ,...vb özel karakterlerle 5 veya daha fazla satır yüksekliğinde dijital metne çevirmeye çalışan python kodu.

### Örnek metin ve çıktısı;

```
metni yazın: halil

#   # ##### #       #   #     
#   # #   # #           #     
##### ##### #       #   #     
#   # #   # #       #   #     
#   # #   # #####   #   #####
```
